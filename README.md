# Тоалет Игрице 🚽🎮

12 free mobile-friendly browser games, packaged as a PWA. Optimized for accessibility — large touch targets, high-contrast dark theme, font-size toggle on every game.

**Live:** https://acosonic.github.io/toiletgames/

## Install

Visit the URL in Chrome (Android) or Safari (iOS) and tap the install prompt.

## Tech

- Vanilla JS/HTML/CSS — no framework, no build step
- Service Worker for full offline support
- All scores saved in `localStorage`
